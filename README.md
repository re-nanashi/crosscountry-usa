## Installation/Development

1. Clone the project <br>
   `git clone https://github.com/re-nanashi/crosscountry-usa.git`

2. Go the directory and run npm install

```
npm install
```

3. Build the game

```
npm run build
```

4. Run the game by opening ./dist/index.html
